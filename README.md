# Muktinath Guest House (MGH) 🏨

A professional hotel management and room booking application built with Flutter. This app allows guests to view rooms, check availability, and make secure bookings for their stay at Muktinath Guest House.

## 🚀 Key Features
- **Real-time Booking:** Browse and book rooms instantly.
- **Digital Payments:** Integrated with **Khalti** and **eSewa** for secure transactions.
- **Location Services:** Google Maps integration for easy navigation to the guesthouse.
- **User Accounts:** Secure login and profile management.

## 🛠 Tech Stack
- **Framework:** Flutter (SDK 3.0.0+)
- **Target SDK:** Android 16 (API Level 36)
- **State Management:** Provider / GetX (Optional based on your setup)
- **Payments:** Khalti Flutter SDK, eSewa Integration.
